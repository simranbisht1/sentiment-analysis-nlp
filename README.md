# Sentiment Analysis on Twitter Data

## Overview
This project performs sentiment analysis on Twitter data using Python.
It classifies tweets as Positive, Negative, or Neutral using VADER
(Valence Aware Dictionary and Sentiment Reasoner).

## Tools Used
- Python, Pandas, Matplotlib
- VADER Sentiment Analysis
- WordCloud

## Results
- Sentiment distribution visualized as bar chart
- WordCloud generated from cleaned tweet text

## Dataset
Public Twitter dataset with labeled tweets.
